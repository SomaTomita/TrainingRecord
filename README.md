# Training Record
 - 筋トレメニュー記録サイトです。
 - twitterで共有し、手軽に記録できます。
 - メニューを記録することで、モチベーション維持やトレーニング方法改善を目指します。

## 目指した課題解決
  ### ペルソナ
- 趣味程度で継続的にジムに通い、筋トレをしている人
- 時間単位である程度やることが決まっているが、日々のトレーニングのメニューにむら(ばらつき)がある人
- 数週間程度の期間で重さやセット数にどのぐらい変化があるか知りたい人
